Visual Studio Code(VS Code)는 Microsoft에서 개발한 강력한 오픈 소스 통합 개발 환경(IDE)입니다. 기본적으로 영어로 제공되지만, 확장 프로그램을 통해 손쉽게 한국어 환경을 구축할 수 있습니다.

이 가이드는 언어 팩 설치부터 설정 확인까지, 모든 과정을 시뮬레이션과 함께 안내합니다.

개발자 IDE 선호도

VS Code는 전 세계적으로 가장 널리 사용되는 도구입니다.

단계별 설치 시뮬레이션

왼쪽의 단계를 클릭하면 오른쪽의 가상 VS Code 화면이 반응하여 설치 과정을 보여줍니다.

1. 확장 프로그램 메뉴 접속

1

화면 왼쪽 '활동 바(Activity Bar)'의 블록 모양 아이콘을 클릭합니다.

Ctrl + Shift + X

2. 한국어 팩 검색 및 설치

2

검색창에 'Korean'을 입력하고 Microsoft 공식 확장 프로그램을 설치합니다.

3. 언어 설정 및 재시작

3

우측 하단 알림창의 'Change Language and Restart' 버튼을 클릭합니다.

수동 설정:
1. Ctrl + Shift + P (명령 팔레트)
2. 'Configure Display Language' 입력
3. 'ko (한국어)' 선택

4. 설정 완료 확인

4

재시작 후 메뉴와 사이드바가 한국어로 변경되었는지 확인합니다.

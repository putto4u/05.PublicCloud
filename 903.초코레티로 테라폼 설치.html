사전 준비 사항 (Prerequisites)

Terraform 설치를 위해 다음 환경이 준비되어야 합니다.

  • 운영체제: Windows 10 이상
  • 터미널: Git Bash (설치됨) 및 PowerShell (관리자 권한)
  • 관리자 권한: 패키지 설치를 위해 필수
참고: Chocolatey(패키지 매니저)를 먼저 설치하면, 이후 Terraform 등 다양한 도구를 명령어 한 줄로 쉽게 관리할 수 있습니다.
Chocolatey Setup
02 / 05

Step 1. Chocolatey 설치

Git Bash가 아닌 PowerShell(관리자 권한)에서 실행해야 합니다.

PowerShell (Admin)
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
설치가 완료되면 PowerShell을 닫고, Git Bash를 관리자 권한으로 새로 실행하세요.
Verification
03 / 05

Step 2. Chocolatey 설치 확인

Git Bash 창에서 아래 명령어로 설치 버전을 확인합니다.

Git Bash
choco --version

버전 넘버(예: 2.x.x)가 출력되면 정상적으로 설치된 것입니다.

Terraform Setup
04 / 05

Step 3. Terraform 설치

이제 Chocolatey를 이용해 Terraform을 간단하게 설치합니다.

Git Bash (Admin 권한 권장)
choco install terraform -y

* -y 옵션은 설치 중간 확인 질문에 자동으로 'Yes'라고 응답합니다.

Complete
05 / 05

Step 4. 설치 검증

Terraform 명령어가 정상적으로 실행되는지 최종 확인합니다.

Git Bash
terraform -version
# 출력 예시 Terraform v1.9.0 on windows_amd64
축하합니다! 이제 인프라 코드를 작성할 준비가 되었습니다.

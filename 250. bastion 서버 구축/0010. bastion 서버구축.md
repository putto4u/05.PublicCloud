  kubectl 설치
1. https://docs.aws.amazon.com/ko_kr/eks/latest/userguide/install-kubectl.html#linux_amd64_kubectl
2. curl -O https://s3.us-west-2.amazonaws.com/amazon-eks/1.34.2/2025-11-13/bin/linux/amd64/kubectl
3. chmod +x kubectl
4. mkdir -p $HOME/bin && cp ./kubectl $HOME/bin/kubectl && export PATH=$HOME/bin:$PATH
5. echo 'export PATH=$HOME/bin:$PATH' >> ~/.bashrc
6. kubectl version 

  eksctl 설치
1. https://docs.aws.amazon.com/eks/latest/eksctl/installation.html  유닉스 설치
2. ARCH=amd64
PLATFORM=$(uname -s)_$ARCH

curl -sLO "https://github.com/eksctl-io/eksctl/releases/latest/download/eksctl_$PLATFORM.tar.gz"

3. tar -xzf eksctl_$PLATFORM.tar.gz -C /tmp && rm eksctl_$PLATFORM.tar.gz
4. sudo install -m 0755 /tmp/eksctl /usr/local/bin && rm /tmp/eksctl
5. eksctl version

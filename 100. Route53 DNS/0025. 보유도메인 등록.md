
보유 도메인 콘솔 등록하기

1. R53
2. 호스팅 영역
3. 호스팅 영역 생성(유료)
4. 도메인 이름 입력, 퍼블릭 호스팅 영역 - 생성
5. 호스트 존 NS Typedml 4개 네임서보 정보 별도 저장
6. 도메일 구매 업체 - DNS 설정 - 네임서버 변경
7. 4개 네임 서버 정보 등록
8. 5~10분후 정상적 변경 여부 확인
   cmd 또는 bash에서
  ```
   nslookup
   > set type=ns
   > putto.ai.kr
   > :
   > 권한 없는 응답
     : 4개 네임 서버 출력
  ```
10. R53에 웹서버에 도메인 연결하기
11. EC2 선택
12. 네트워크 및 보안
13. EIP - EIP 주소 할당 - 할당
14. 작업 - EIP 주소 연결
15. 인스턴스 선택 - 연결할 프라이빗 IP 선택 - 연결
18. 인스턴스 페이지에서 ec2 와 EIP 확인
19. 서비스-네트워킹 및 콘텐츠 전송 - R53 다시 이동
20. R53-호스팅 영력 - 도메인 이름 클릭
21. 상단 레코드 세트 생성 - 레코드 세트 생성 항목 이름 : www 입력, 값: EIP 입력 - 생성
22. 웹브라우저에 도메인 입력 홈페이지 확인

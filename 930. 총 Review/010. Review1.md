03. 네트워크  :  연결, 소통   VS   보호(내부자원(정보)보호)
    연결  -  인터넷과 연결해서 서비스 목적, 각 APP/서비스   연결
    보호  -  내부 중요 정보를 보호 목적이 크게 상승  ->  연결과 더불어 네트워크 방어/은폐 목적 ->  퍼블릭 네트워크와 프라이빗 네트워크 개념 -> 구현 pub IP network vs private IP/서브넷,vlan

04. 네트워크 중요 개념  - OSI 7Layer 구현 목적 표준 제시  -> 프로토콜
    ----------------- 데이터 라인  ->  ...[헤더][헤더][헤더][IP/port헤더][Mac헤더][데이터]
    1) 물리적 연결
    2) 소실 없는 데이터 소프트웨어적 전송기술  ---  Mac 주소 
    3) 각 node에 대한 구분 -> IP 개념
    4) 정해진 목적대상에 목적된 데이터 가도록 마련된 별도의 길 -> Route
    5) 여러가지 목적의 프로그램 동시에 써야하는 상황 -> port로 분리(목적,대상 프로그램)
    6) 하나의 프로그램 내에서도 여러가지 일을 해야함. 여러 사람이 접속 -> 멀티 유저, 멀티 태스팅  -> 세션
    7) 여러 기술들의 표현 방식이 다른 것에 대한 분류 -> 표현방식
    8) 최종적 사용자 데이타가 전송
   
    9) 헤더에 대한 지식 - 프레임(L2)  >   패킷 구분(L3,L4)

글로벌 인프라 계층 구조

지리적 영역 (Geography)

데이터 레지던시 및 규정 준수 경계를 보존하는 별도의 시장 단위입니다. 일반적으로 두 개 이상의 리전을 포함합니다. (예: 한국, 미국)

리전 (Region)

지연 시간(Latency)이 정의된 경계 내에 배포되고, 전용 초저지연 네트워크를 통해 연결된 데이터 센터의 집합입니다.

가용 영역 (Availability Zone)

Azure 리전 내에서 물리적으로 분리된 위치입니다. 각 영역은 독립적인 전원, 냉각 및 네트워킹을 갖춘 데이터 센터로 구성됩니다.

컴퓨팅 서비스 스펙트럼

Virtual Machines

IaaS: Windows/Linux 서버 가상화. OS에 대한 완전한 제어권 제공.

App Service

PaaS: 인프라 관리 없이 웹 앱 및 API를 구축/배포하는 관리형 서비스.

AKS (Kubernetes)

마이크로서비스 아키텍처를 위한 완전 관리형 컨테이너 오케스트레이션.

Azure Functions

Serverless: 이벤트 기반으로 코드를 실행하며, 사용한 만큼만 과금.

Azure Virtual Desktop

[차별화] 유일하게 Windows 10/11 멀티 세션을 지원하는 클라우드 VDI.

Azure Arc

온프레미스, 타 클라우드 등 모든 인프라로 Azure 관리를 확장.

스토리지 솔루션 분류

Blob Storage

비정형 데이터(이미지, 영상, 로그 등)를 위한 대규모 객체 스토리지입니다.

AWS S3 대응

Disk Storage

Azure VM을 위한 고성능 블록 스토리지입니다. (SSD/HDD 선택 가능)

AWS EBS 대응

Azure Files

표준 SMB 프로토콜을 통해 접근 가능한 완전 관리형 파일 공유 서비스입니다.

AWS EFS 대응

(True Hybrid)
